# Boothy (RapidTetherRAW)

Boothy (working title: RapidTetherRAW) is a desktop application designed for **Canon EOS 700D** tethered shooting, RAW processing, and exporting.
It is a fork of RapidRAW, optimized for a minimal studio workflow, focusing on speed and simplicity.

## Key Features
- **Tethered Shooting**: Seamless connection and control for Canon EOS 700D via EDSDK.
- **RAW Processing**: High-performance RAW development powered by the RapidRAW engine.
- **Minimalist UX**: Streamlined "Shoot → Review → Process → Export" workflow for studio environments.
- **Session Management**: Efficient handling of shooting sessions with auto-apply presets.
