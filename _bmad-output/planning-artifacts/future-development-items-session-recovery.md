# 추후 개발사항: 세션 복구 및 재개 개선

작성일: 2026-03-31

## 목적

이 문서는 프로그램 오류, 재부팅, 또는 예기치 않은 종료 이후에
손님이 촬영하던 세션을 다시 불러오고 이어갈 수 있는 복구 경험에 대한
**추후 개발사항**을 기록한다.

## 현재 판단

- 현재 단계에서는 이 기능을 즉시 반영하지 않는다.
- 먼저 MVP에서 기본 촬영, 저장, 미리보기, 결과 확인이 정상 동작하는지 검증한다.
- 기본 기능 검증이 끝난 뒤, 세션 복구와 재개 경험을 별도 개선 과제로 검토한다.

## 추후 개발사항

- 앱 종료 또는 재부팅 이후에도 기존 세션을 다시 열 수 있는 복구 흐름
- 현재 세션에 저장된 사진을 다시 로드해서 확인할 수 있는 흐름
- 최종 결과물이나 미리보기 상태를 다시 불러오는 흐름
- 같은 세션에서 촬영을 이어갈 수 있도록 하는 복구 흐름
- 운영자 개입이 필요한 경우를 위한 bounded recovery 흐름

## 반영 금지 조건

- 현재 진행 단계에서는 **사용자의 명시적 허가 없이 절대 반영하지 않는다.**
- 이 문서는 구현 지시가 아니라 추후 검토 대상 기록이다.
- MVP 검증 전에는 PRD, 에픽, 아키텍처, 구현 스토리에 이 내용을 자동 반영하지 않는다.

## 정리

이 항목은 **추후 개발사항**으로만 관리한다.
현재 단계에서는 기본 기능 확인에 집중하고,
세션 복구 개선은 별도 승인 이후에만 진행한다.
