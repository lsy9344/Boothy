## 무인 셀프사진관 브랜드 컨텍스트 (Agent Knowledge Base)

이 문서는 “내가 운영하는 무인 셀프사진관 브랜드/매장”에 대해 에이전트가 **정확하고 일관된 안내·운영지원**을 하기 위한 기준 문서입니다.

### 문서의 목적(중요)
- 이 파일은 “지식파일(Single Source of Truth)”입니다. 이 파일 하나로 아래 산출물을 에이전트와 함께 만들어낼 수 있어야 합니다.
  - 지점장/운영자에게 제공할 운영 매뉴얼, 교육 자료, 체크리스트
  - 고객에게 제공될 챗봇(FAQ/정책/이용 안내)의 지식 데이터베이스
  - 장애 대응 플레이북(원격지원 SOP), 공지/문자 템플릿, 운영 리포트 템플릿
- 따라서 이 문서는 **정책/가격/운영 방식의 기준**을 명확히 기록하고, 애매한 부분은 “확정 필요”로 남긴 뒤 질문을 통해 빠르게 확정하는 것을 목표로 합니다.
- 문서의 독자(권장 구분)
  - 고객(공개): 가격/이용 흐름/FAQ/대표번호/링크
  - 지점장/운영자(내부): 운영 SOP/장애 대응/재고·청소/환불·보상 판단 기준
  - 오너/권한자(내부-고위험): 보안/권한/계정 접근, 예외 승인 정책

### 작성 원칙
- **추측 금지**: 불확실하면 “확인 필요”로 답하고, 필요한 질문을 먼저 합니다.
- **시간 정보는 날짜와 함께**: 가격/정책/프로모션/운영시간은 `적용 시작일(필수)`·`종료일(있으면)`을 기록합니다.
- **보안/개인정보**: 비밀번호, API 키, 카드번호, 주민번호 등 비밀정보는 절대 쓰지 않습니다. 대신 “보관 위치(예: 1Password 항목명)”와 “접근 절차”만 기록합니다.

### 정보 등급(권장)
- 공개 가능: 가격/이용 흐름/FAQ/대표번호/링크 등 고객 안내용 정보
- 내부 공유(지점장/운영자용): 운영 체크리스트/장애 대응/재고·청소 루틴/환불·보상 판단 기준 등
- 문서에 값 금지(비밀): 도어락 비밀번호, 원격 접속 ID/비번, 계좌번호 등 *(필요 시 “보관 위치/접근 절차”만 기록)*

### 문서 운영(필수)
- 기준일: 2026-01-29 *(정책/가격/프로모션은 각 항목에 별도 기준일 표기)*

  - 정책/가격/보상: 오너(또는 권한자)
  - 고객 응대/운영 예외: 운영자

## 4) 에이전트가 해야 하는 일/하면 안 되는 일 (필수 확정)
### 운영 전략 및 리스크 관리 (Founder Decision)
- **수동 스케줄링 리스크**: 평일 쿠폰/이벤트의 다음 타임 수동 블락 방식은 현재 유지함. (이유: 추후 개발될 자동화 구독 시스템으로 해결 예정)
- **현장 계좌이체 기록**: 이럴 경우가 거의 없어 별도의 내부 기록을 남기지 않는 현안 유지. (리스크 감수)
- **폴더명 개인정보 노출**: `예약자명+뒤4자리` 폴더명 생성 방식 현안 유지.

### 해야 하는 일(예시)
- 메뉴/상품(시간권/패키지) 안내, 예약/이용 방법 안내, 위치/주차/찾아오는 길 안내
- 기본 CS(자주 묻는 질문), 규정 기반 환불/변경 가능 여부 안내
- 장비 사용법/문제 해결 1차 가이드(안전한 범위)
- 촬영/내보내기 매니저 실행 안내 + 오류 시 원격지원 동의/에스컬레이션

### 하면 안 되는 일(예시)
- 가격/정책을 임의로 약속하거나 예외 승인하기
- 환불/보상 확정(권한자 승인 없이)
- 개인정보 과다 수집/저장(민감정보 요청 포함)

## 8) 다운로드/데이터/개인정보 처리 (중요)

### 10.3.4 `다비 촬영 매니저` (촬영 단계) 정밀 대응 SOP (2026-01-29 업데이트)
> **증상**: 셔터를 눌러도 반응이 없음, "카메라 감지 실패", 프로그램 멈춤

    -   창이 떠 있는데 촬영이 안 된다면: 원격으로 **필터를 다른 것으로 한 번 바꾼 뒤** 다시 시도해봅니다.
2.  **2단계: 소프트웨어 재시작**
    -   실행 중인 `Adobe Lightroom Classic`을 완전히 종료합니다.
    -   바탕화면의 `다비 촬영 매니저`를 다시 실행합니다.
    
#### [대체] 라이트룸에서 수동으로 촬영 시작(운영자/지점장 전용)
> **상황**: 고객이 입실 후 `다비 촬영 매니저`를 실행했는데 `자동화 오류` / `카메라를 감지할 수 없습니다` / 기타 경고창 등으로 촬영 시작이 막힌 경우, 운영자가 **원격으로** 라이트룸에서 촬영을 직접 시작합니다. *(고객에게 라이트룸 메뉴 조작을 상세히 시키지 않음)*

1.  **스텝 1: 경고창/프로그램 정리 후 라이트룸 실행**
    -   열려 있는 경고창(오류 팝업)과 관련 프로그램을 모두 닫습니다.
    -   `Adobe Lightroom Classic`을 실행합니다.
2.  **스텝 2: 연결 전송된 촬영 시작 메뉴 진입**
    -   라이트룸 좌측 상단 `파일(File)` 메뉴 클릭
    -   `연결 전송된 촬영(Tethered Capture)` → `연결 전송된 촬영 시작(Start Tethered Capture)` 클릭
3.  **스텝 3: 세션 이름 입력**
    -   새로 뜨는 창에서 `세션 이름`에 `예약자명+휴대전화뒤4자리`를 입력합니다. (예: `홍길동1234`)
    -   확인/시작을 눌러 촬영을 진행합니다.
    
> **후속**: 촬영이 정상 진행되면 촬영 종료 시에는 원칙대로 `다비 내보내기 매니저`로 변환을 진행합니다. 변환까지 자동화가 실패하면 `10.3.6`의 “라이트룸 수동 내보내기”로 전환합니다.

3.  **3단계: 하드웨어 강제 리셋 (스마트 플러그)**
    -   위 방법으로 해결되지 않거나, 프로그램이 완전히 먹통인 경우
    -   **"원격 전원 콘센트(IoT 스마트 플러그)"** 앱을 이용해 카메라/PC 연결 전원을 껐다가 켭니다.
4.  **4단계: 최후의 수단 (현장 해결 불가 시)**
    -   인터넷 장애 등으로 원격 접속조차 불가능하거나, 위 방법으로도 해결이 안 될 때
    -   **고객 귀가 조치**: "죄송합니다. 기기 문제로 현재 촬영이 어렵습니다. 우선 귀가하시면 제가 직접 원본/보정본을 완벽하게 챙겨서 보내드리겠습니다."라고 안내.
    -   **사후 처리**: 오너가 현장 방문 또는 나중에 원격으로 사진을 확보하여 메일 전송/인화 배송 등 직접 서비스.

### 10.3.5 `다비 내보내기 매니저` (변환 단계) 정밀 대응 SOP (2026-01-29 업데이트)
> **핵심 메커니즘**: 촬영실 PC 로컬에 저장 → 네트워크 공유 폴더를 통해 셀렉실 PC에서 읽어옴.
> **경로 원칙**: 무조건 `바탕화면 -> 사진저장폴더` 경로여야 함.

1.  **변환 멈춤/오류 시 (재시작 원칙)**
    -   **조치**: 프로그램을 종료 후 재실행합니다.
    -   **옵션 선택**: "새로 시작하기" 등의 버튼이 있다면 **[덮어쓰기]** 또는 **[새로 시작]**을 선택합니다. (기존 파일을 덮어쓰며 완벽하게 다시 생성하는 것을 추천)
2.  **셀렉실에서 사진이 안 보일 때**
    -   **원인 1 (폴더명 불일치)**: 촬영 때 입력한 이름과 내보내기 때 입력한 이름이 다르면 폴더가 2개 생성됩니다. (예: `홍길동1234` vs `홍길동`)
        -   *해결*: 촬영실 PC 바탕화면의 `사진저장폴더`에 들어가서 실제 생성된 폴더명을 확인합니다.
    -   **원인 2 (네트워크 오류)**: 셀렉실 PC에서 촬영실 공유 폴더 접근이 끊긴 경우.
        -   *해결*: 인터넷 연결 확인 및 촬영실 PC가 켜져 있는지 확인.
3.  **다음 타임과 시간이 겹칠 때 (동시 작업)**
    -   **원칙**: 내보내기 프로그램이 돌아가는 도중에 `다비 촬영 매니저`를 켜서 다음 팀 촬영 준비를 **동시에 할 수 있습니다.**
    -   **주의**: 내보내기 프로그램이 완료되기 전에 종료하면 안 됩니다. (창을 최소화하거나 뒤로 보내두고 촬영 세팅 진행)

### 10.3.6 [비상용] 라이트룸 수동 내보내기(Manual Export) 가이드
> **상황**: `다비 내보내기 매니저` 매크로 오류 시, 운영자가 원격으로 직접 라이트룸을 조작해 사진 내보내기 할 때.
> *(원칙: 고객에게 라이트룸 메뉴 조작을 상세히 시키지 않음 — 운영자/지점장이 원격으로 처리)*

1.  **사진 전체 선택**
    -   라이트룸 하단(필름스트립)에 나열된 사진 중 **아무거나 하나를 클릭**합니다. (만약 라이트룸 하단에 가로로 길게 나열된 사진들이 보이지 않는다면 하단 정 중앙을 보면 상단을 가리키는 화살표가 희미하게 보일겁니다. 이걸 클릭하면 가로로 나열된 사진들이 보여집니다.)
    -   `Ctrl + A` (맥: `Cmd + A`)를 눌러 **모든 사진을 선택**합니다.
    -   전체 선택은 “선택이 안 된 것처럼” 보일 수 있으나, 하단 필름스트립 사진들이 **조금 더 밝게** 보이면 선택된 상태입니다.
2.  **내보내기 창 열기**
    -   하단 필름스트립의 사진 중 아무거나 위에서 **마우스 오른쪽 버튼 클릭** → `내보내기(Export)` 클릭.
3.  **내보내기 위치 설정 (가장 중요)**
    -   내보내기 창 최상단 `내보내기 위치` 항목에서 **[특정 폴더(Specific Folder)]**를 선택합니다.
    -   **경로 선택**: `선택...` 버튼 클릭 → `바탕화면` → **`사진저장폴더`** 를 찾아 선택합니다.
    -   **하위 폴더**: `하위 폴더에 넣기` 체크박스 체크.
    -   **폴더 이름**: `예약자명+휴대전화뒤4자리` 입력 (예: 홍길동1234).
4.  **파일 이름 지정(리네이밍)**
    -   `파일 이름 지정` 메뉴에서 `바꿀 이름` 체크박스를 체크합니다.
    -   `사용자 정의 이름 - 시퀀스`를 선택합니다.
    -   `사용자 정의 텍스트`에 `필터`를 입력하고, `시작 번호`를 `1`로 설정합니다.
5.  **파일 설정**
    -   `이미지 형식`: `JPEG`
    -   `품질`: `90`
    -   `파일 크기 제한(Limit File Size To)` 체크박스는 **체크하지 않습니다.**
6.  **이미지 크기 조정**
    -   `크기 조정하여 맞추기(Resize to Fit)` 체크박스를 체크합니다.
    -   `너비 및 높이(Width & Height)`를 선택합니다.
    -   `확대 안함(Don't Enlarge)` 체크박스는 **체크 해제**합니다.
    -   `너비` `2456` / `높이` `4000` / 단위 `픽셀(px)`로 설정합니다.
    -   `해상도` `160` / 단위 `인치당 픽셀(ppi)`로 설정합니다.
7.  **출력 선명하게 하기**
    -   `선명하게 하기(Sharpen For)` 체크박스를 체크합니다.
    -   대상은 `화면(Screen)`, 양은 `표준(Standard)`으로 선택합니다.
8.  **그 외 메뉴**
    -   위에서 언급하지 않은 모든 메뉴는 **수정하지 않고 그대로 둔 채** 다음으로 넘어갑니다.
9.  **실행**
    -   우측 하단 [내보내기] 버튼 클릭.

### 10.3.7 이메일 전송/인화 장애 대응

### 10.7 안전/긴급 상황 대응(권장)

- 부상/실신/화재/연기/가스 냄새 등 위험: `119` 신고 안내 → 가능하면 안전한 장소로 이동 → 운영자/오너에게 연락

- 참고: 야간/긴급 상황을 “자체 해결”로 기재한 경우가 있으나 안전/법적 리스크가 커서, 본 문서의 원칙(112/119 우선)을 적용 권장
